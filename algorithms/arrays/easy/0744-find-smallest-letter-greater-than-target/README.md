# 0744. Find Smallest Letter Greater Than Target

🔗 Problem Link:
https://leetcode.com/problems/find-smallest-letter-greater-than-target/

## 🚀 Approach
- Find ASCII code of target
- Loop through char []
  - find ASCII each char
  - check char code with target code
  - if it char code > target code
  - return char
- return first char from char []
